- cj_empY.csv (임직원 : 설 연휴 포함)

-> num_category 열은 제거하는 게 성능이 더 좋게 나옴 (현재 데이터프레임엔 일단 포함시킴)
#### 카테고리 (개수로 반영)

-> 비임직원인 경우만 num_category 열은 제거하는 게 성능이 더 좋게 나옴 (현재 데이터프레임엔 일단 포함시킴)
